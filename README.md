# Student Performance Prediction (End-to-End ML)

## Problem Statement
Predict student academic performance using demographic and academic data.

## Tech Stack
Python, Pandas, NumPy, Scikit-learn, Flask

## Project Structure
- src/ → core pipeline
- notebook/ → experimentation
- artifacts/ → trained models
- application.py → Flask app

## How to Run
1. Install requirements
2. Run application.py
3. Access local API

## Results
Achieved optimized regression performance using feature engineering and tuning.
